# TV Time Clone – تعليمات التشغيل

## المتطلبات
- Node.js مثبت عندك (تقدر تتأكد بكتابة `node -v` في الـ terminal)
- API key من TMDb (مجاني)

## خطوات التشغيل

1. فك ضغط ملف tv-time-clone.zip

2. افتح Terminal / CMD جوه المجلد وشغل:
```
npm install
```
ده هيثبت كل المكتبات المطلوبة (Express, lowdb, bcryptjs, node-fetch, express-session)

3. حط الـ API key بتاعك. فيه طريقتين:

   **الطريقة الأسهل (مباشرة في الكود):**
   افتح ملف `routes/shows.js` ولاقي السطر:
   ```
   const TMDB_API_KEY = process.env.TMDB_API_KEY || "YOUR_API_KEY";
   ```
   وغيّر `YOUR_API_KEY` بالـ key بتاعك.

   **الطريقة الأفضل (متغير بيئة):**
   قبل ما تشغل السيرفر، اكتب:
   ```
   set TMDB_API_KEY=your_key_here      (على ويندوز)
   export TMDB_API_KEY=your_key_here   (على ماك/لينكس)
   ```

4. شغل السيرفر:
```
npm start
```

5. افتح المتصفح على:
```
http://localhost:3000
```
هيوجهك لصفحة تسجيل الدخول تلقائيًا. اعمل حساب جديد وابدأ استخدام التطبيق.

## هيكل المشروع
```
tv-time-clone/
  server.js          -> نقطة انطلاق السيرفر
  db.js              -> إعداد قاعدة البيانات (ملف database.json بيتعمل تلقائيًا)
  routes/
    auth.js          -> تسجيل الدخول / حساب جديد / خروج
    shows.js          -> البحث عن مسلسلات + إضافة لقائمتي + تتبع الحلقات
  public/
    login.html        -> صفحة الدخول
    index.html         -> الصفحة الرئيسية (بحث + مسلسلاتي)
    show.html           -> صفحة تفاصيل مسلسل معين (المواسم والحلقات)
    app.js, show.js     -> منطق الفرونت اند
    style.css           -> التنسيق
```

## ملاحظة مهمة
قاعدة البيانات هنا عبارة عن ملف `database.json` بيتعمل تلقائيًا أول ما تشغل السيرفر.
ده كويس للتعلم والتجربة، بس في مشروع حقيقي/production لازم تستخدم قاعدة بيانات حقيقية
زي PostgreSQL أو MySQL بدل الملف ده.

## الخطوة الجاية (اختياري)
لو عايز تكمل، ممكن نضيف:
- إحصائيات (عدد الحلقات/الساعات اللي اتفرجت عليها)
- فيتشر اجتماعي (متابعة أصحاب)
- إشعارات لما حلقة جديدة تنزل
